Strategy of branch management

we have master,pre,test branch in company for example

  1. checkout master_hotfix branch from master
  2. merge master_hotfix to test branch and test pass
  3. merge master_hotfix to pre branch and test pass
  4. checkout master_release branch from master and then master_hotfix merge to it
  5. release master_release branch,and finally merge back to master
