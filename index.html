函数依赖

  1. 记A->B 表示A函数决定B(B依赖于A)
  2. 对于 A->B，如果能找到 A 的真子集 ‘A’，使得’A’-> B，那么 A->B 就是部分函数依赖，否则就是完全函数依赖。
  3. 对于 A->B，B->C，则 A->C 是一个传递函数依赖。
  4. 如果 {A1，A2，… ，An} 是关系的一个或多个属性的集合，该集合函数决定了关系的其它所有属性并且是最小的，那么该集合就称为键码。

事务管理

  • 定义:事务处理用来维护数据库的完整性，保证成批的 SQL 语句要么全部执行，要么全部不执行
  • 事务四大特性：ACID
    • 原子性:操作要不全部成功,要不全部失败回滚
    • 一致性:事务执行前后都必须处于一致状态
    • 隔离性:并发事务互不干扰
    • 持久性；事务提交成功,对数据库的数据改变是永久的

多线程

1.并发与并行

  • 并发:指两个或多个事物在同一个时间段内发生
  • 并行:指两个或多个事物在同一个时刻发生(同时发生)

2.线程与进程

  • 进程:进程是程序的一次执行过程，是系统运行程序的基本单位.
  • 线程:线程是进程中的一个执行单元，负责当前进程中程序的执行，一个进程中至少有一个线程.
    (简要来说,进程就是抢占cpu的资源,然后调用线程去利用这些资源)

集合

Java中的集合体系理论

  • 数组:存放数据的容器,同一个数组中只能存放同一种类型的数据(元素),可以存储是引用类型或者基本数据类型的数据.
  • 集合:存放数据的容器,只能存储引用数据类型,在无泛型规定时可以添加任意的引用数据.
  • 集合的分类:单列集合(collection),双列集合(map),单列集合分为list和set
  • Iterator(java.util)是集合最顶层的接口,用于迭代访问集合中的元素
    • boolean hasNext()
    • E next()
    • void remove()
  • Iterable(java.lang)是集合的顶级接口之一,让该接口的子接口或者实现类具有可迭代性
    • Iterator iterator() 返回一组T类型元素进行迭代的迭代器,用于迭代遍历集合中的元素
