redis

Created2023-03-11|Updated2025-10-08|middleware
|Post Views:

本地安装

应用场景

1、查询缓存，注意缓存穿透和雪崩问题
2、短信登录，Redis共享session应用
3、优惠卷秒杀，Redis计数器、Redis分布式锁、Redis三种消息队列
4、排行版功能，基于zset，设置分数
5、附近商户，GeoHash存放ip信息

数据类型

数据类型 可以存储的值 操作
STRING 字符串或数值 字符串操作,数字类型的自增自减
LIST 列表 就是队列
SET 无序集合 去重，集合计算，是否存在某元素判断
HASH 包含键值对的无序散列表 存k->Map(k,v) 可存一个java bean类的对象
ZSET 有序集合 可为元素设置分值，根据分值进行排序，查询分数范围的对象
Geospatial 地理位置 通过Java将经纬度数据导入redis，之后即可通过经纬度计算距离等
Hyperloglog 基数(两个数据集中不重复的元素) 适用于快速合并数据去重，允许一定误差值，效率很高(网站访问量统计)
Bitmap 位存储 二进制记录，适用于统计员工打卡
