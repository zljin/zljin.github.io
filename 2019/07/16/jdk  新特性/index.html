JDK8新特性

JDK8 新特性

lambda表达式

  • lambda就是对匿名内部类的简化 (形参1…)-> 返回值
  • 匿名内部类对象核心:是方法的重写,需要重写的方法有且只有一个
  • 方法的核心:形参列表和函数体语句
  • lambda 表达式的局部变量可以不用声明为 final,默认final修饰
  • lambda 表达式只能引用标记了 final 的外层局部变量
  • lambda表达式可以自动匹配数据类型

函数式接口

  • 函数式接口(Functional Interface)就是一个有且仅有一个抽象方法，但是可以有多个非抽象方法的接口。

  • 函数式接口可以被隐式转换为 lambda 表达式。

  • Predicate 传入数据得到boolean值

    • 抽象方法 boolean test(T);

  • Supplier T get();

  • Consumer

    • void accept(T t);

  • Function<T,R> 运行 活动

    • R apply
    • andThen(after) 指定后继操作
    • compose(before) 执行前置操作

方法引用(::)

  • 通过方法的名字指向一个方法

  • 借助其他类现有的方法，去重写 抽象方法

    • 首先写一个泛型函数式接口

    • 然后写一个学生类

    • 开始进行方法引用的操作

Stream

  • 链式编程 对Array list set 增强类
  • Stream 使用一种类似用SQL语句从数据库查询数据的直观方式来提供一种对 Java 集合运算和表达的高阶抽象
  • 这种风格将要处理的元素集合看作一种流， 流在管道中传输， 并且可以在管道的节点上进行处理， 比如筛选， 排序，聚合等。

元素流在管道中经过中间操作（intermediate operation）的处理，最后由最终操作(terminal operation)得到前面处理的结果。

  • 具体用法

默认方法

  • 接口中可以定义有方法体的方法(可以包含方法的实现),如果是非静态,必须用default修饰

JDK7 特性

  • try-with-resources 语句 //这样写不用关流

  • 泛型简化,菱形泛型

  • switch 语句可以用字符串

  • 异常的多个catch合并,每个异常用或|
