ajax&jquery用法

1.ajax简介

  • Ajax是一种在无需重新加载整个网页的情况下，能够更新部分网页的技术
  • AJAX = 异步 JavaScript 和 XML
  • Ajax运行原理:通过在后台与服务器进行少量数据交换，AJAX 可以使网页实现异步更新。这意味着可以在不重新加载整个网页的情况下，对网页的某部分进行更新

2.XMLHttpRequest

  • XMLHttpRequest对象用于在后台与服务器交换数据。这意味着可以在不重新加载整个网页的情况下，对网页的某部分进行更新。

  • 使用流程:(1)创建此对象(2)请求(3)响应

  • 请求:Request.open(‘post’,url,async) && Request.send(str)

    • method:请求方法 ‘post’
    • url:文件在服务器上的位置
    • async:true异步
    • str:参数用于post请求,get不需要写param

  • 获取响应数据request.responseText

  • ajax状态码request.readyState

    • 0: 请求未初始化
      • 1: 服务器连接已建立
    • 2: 请求已接收
    • 3: 请求处理中
    • 4: 请求已完成，且响应已就绪

json

  • json字典相当于java的map
  • json数组相当于数组存了map,字典

jackson

  • Jackson是一个简单基于Java应用库，Jackson可以轻松的将Java对象转换成json字符串和xml文档，同样也可以将json、xml转换成Java对象

  • 对象转字符串

  • 字符串转对象

3.基于js的post实例（不常用）

  • First:前端表单页面

  • Second:ajax的js代码(返回json格式的数据)

  • Third:服务器处理

4.基于jQuery用法（常用）

  • JQuery的get讲解(post改方法名即可)

  • JQuery表单序列化作用:params可以简化
