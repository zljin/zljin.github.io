js简介

ECMAScript为js实现的基础
ECMAScript仅仅是一个描述,定义了脚本语言的所有属性,方法和对象,js以它来作为功能的基准.

js组成=核心(Es)+DOM+BOM

js基础语法

  • 变量声明: var

  • 数据类型:

    • 原始类型(值类型,字面值):

      • String,Number,Boolean,Undefined(未初始化的变量),Null(对象的占位符)

    • 引用类型:(继承父类Object)

      • var obj=new Object()
      • Boolean,Number,Array,String,Date,Math,RegExp.

  • 类型转换:

    • String,Number,Boolean的原始值为伪对象(有具体的属性和方法)
    • toString() parseInt()/parseFloat()
    • 强转:Boolean(0) Number(“12”) String(true)

  • 运算符与表达式

    • 比较运算符

      • ===:全等:类型与值都要相等

    • 空运算 a href=”javascript:void(0);”

    • 类型运算符
