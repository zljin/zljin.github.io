Java基础

Created2019-01-22|Updated2026-01-22|Java
|Post Views:

JDK LTS: 1.8、17、21

基础语法

数据类型：分为基本数据类型、包装类型、引用类型

基本数据类型：
byte char boolean
short int long float double

包装类型：
将简单的基本数据类型表示为一个类,好处就是会有一些类方法，如int->Integer

引用类型：
如String, array

关键字：定义的一些具有特殊用途的单词

标识符：给程序中的变量、类、方法命名的符号

运算符：给程序进行数据的运算,赋值,比较的符号

流程控制：分支结构和循环结构

面向对象

类是一个具有相同特征和相同行为的抽象描述,对象是通过类创建具体实例

Object

泛型

将类型参数化。定义在类,接口,方法时这个类型形参将在创建对象,调用方法时动态的指定

  1. 适用于多种数据类型执行相同的代码,提高代码复用性和扩张性
  2. 提供了编译时类型安全检查

反射

在运行时能够动态的获取类的信息和动态调用对象的方法和属性的能力。优点是扩展性强,比较常用的例子就是sql.driver，常用于框架设计

注解

反射可以通过获取标注内容信息实现自定义功能如被标注的方法,类

异常

何时何地报出什么类型的错误,用于程序的健壮性,容错性和系统的可维护

集合

Iterator 迭代器是单列集合的统一接口，采用迭代器设计模式,将集合的存储结构与遍历逻辑分离,且迭代器可以在迭代的时候安全删除，是并发安全的

双向迭代器

集合的比较器排序Comparator

集合的自然排序,实现Comparable接口重写compareTo方法

Deque 双端队列实现stack和queue

Map遍历

流

以应用程序为主体,往应用程序输入为输入流,从程序输出为输出流

序列化：实现Serializable 接口，并定义serialVersionUID, transient修饰的属性不会序列化

1、 序列化：将对象以字节序列的形式持久化到一个文件中。
2、反序列化：解析字节序列的文件并生成对象。(将对象进行冰冻和解冻,常用于流数据的传输等)

文件拷贝方法
