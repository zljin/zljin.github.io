daily meeting say what?

Yesterday,
For Ticket 999, I implemented the feature that new api can export data as excel. And,it’s ready to test.

Today,
I’ll setup this job in aws and do a self-test.
After that,I’ll move on to ticket 992.
I’m still working on the backend feature that exports adverse news as Word documents.
Ideally, that should be ok by next Monday.

Any block?
No blockers at the moment. That’s all.

I’m currently blocked on doing sth/sth,
So I’ll coordinate with the frontend team to align on the new sia API contract.

How to talk about project progress

We’ve already set targets.
We haven’t created a budget yet.
Have you defined the project goal yet?
We still haven’t defined roles.

Talking About problems difficulties

We’re having some trouble with the new production line.
There are several problems that need to be addressed.
We need to deal with this problem immediately.
It can be quite hard to wirte business report in english.
We find it hard to reproduce this case.
