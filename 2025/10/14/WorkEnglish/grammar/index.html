名词

分普通名词，专有名词，可数与不可数名词
冠词：名词前面要加，特指the,泛指a, 复数泛指/专有名词/泛指概念不需要

代词

人称代词：

主格 宾格
I me
we us
you you
he/she him/her
they them
it it

物主代词:

形容词性物主代词 名词性物主代词
It’s my computer. The computer is mine.
our ours
your yours
his/her his/hers
its its
their their

名词所有格: Mike’s PC

指示代词：this/these(指较近的事务) that/those(较远的事务)

不定代词
some time / any issues?
many & a few (修饰可数) / much &a little(修饰不可数)
either Kafka or RabbitMQ (二选一)
neither scalable nor secure (两个都不)
each other (两三者以上)

形容词

more/most 足够用了，主要修饰名词

This method is more safe than the other one.

介词

为前置词用来提供附加信息如时间/空间/方式/原因/目的等，介词不单独存在，而是“动词/形容词 + 介词”成对出现,记一些搭配用法

副词

副词是用来修饰动词、形容词、其他副词，或整个句子的词。

控制沟通语气，避免冒犯
Honestly, I’m not convinced by this approach. (表达态度)
Ideally, every PR should include unit tests and updated docs. (表达态度)
Actually, I think the latency impact is underestimated. (委婉矫正)
Critically, this affects user data security. (强调重点)
This change removes a deprecated library. In addition, it eliminates a critical security vulnerability. (表示递进)

高效表达逻辑关系
The test coverage is low. Therefore, we can’t merge yet. (推进共识)

表达如程度，频率等等
we usually have a stand-up meeting every Monday.
We need to communicate this change effectively to all stakeholders.

As the backend lead,I’ll own the migration plan.
As traffic increased, latency spiked.
As security is critical, all data must be encrypted.

邮件常用开场：
Just to clarify... / To recap... / As discussed in yesterday’s sync...

动词

谓语动词

具有时态与语气的能力

现在完成时：
We’ve finalized the API Spec.

虚拟语气：
If we used Kafka here,lantency would drop by 30%.

系动词

表赋值含义，如 be动词

助动词

辅助谓语动词做谓语

I have done it. I don’t think so. I can kill the process.

非谓语动词

做非谓语的句子成分,可以简化从句，有不定式，动名词，现在分词，过去分词四种形式
介词后加动名词是铁律 ，分词 (记下不规则动词表)，主要是固定搭配

We decided to migrate to AWS.
Deploying without tests is risky.
His job is teaching English.

The talk was interesting. I’m interested in this project.
A sleeping baby, and a broken window.

当分词短语放名词后相当于一个定语从句，优势：简化定语从句，做状语
The man standing (= who is standing) there is my teacher.
The book written (= which was written) by him is popular.

独立主格做状语
通过 名词/代词+非谓语成分构成的独立结构，本身不是句子，用来修饰主句，描述背景/原因/条件

当分词的逻辑主语不是句子主语时:
The deadline approaching, the team is working around the work.

当分词的逻辑主语是句子主语时,不需要自带“主语”:
Hearing the good news, she jumped with joy.

简单句与从句

简单句：主+谓 结构，是句子核心，下面是分类
1、主语+不及物谓语动词：Baby Sleep
2、主语+及物谓语动词+宾语：Baby like mother
3、主语+谓语动词+间接宾语+直接宾语：She teaches you english
4、主语+谓语动词+宾语+宾补：She consider you smart
5、主语+系动词(赋值)+表语：She is beautiful.She smells nice.

从句：充当主句的某个句子成分(从句可以理解为主句里面的子句，本身也是一个句子，肯定有主谓结构) 主从时态要一致

从句格式：主句 + 引导词/连接词 + 从句

主语从句:
It’s obvious that the rabbit ate the carrot.

宾语从句：
I think that we’ll have plenty of time

定语从句：
She ate the largest carrot that I bought.
The rabbit ate the carrot, which was not surprising. which修饰逗号前的整个句子，非限定性定语从句

表语从句：
the problem is that the baby is hungry

状语从句：用一个完整句子来干副词的活

If we don’t add rate limiting, the API will be abused.
Although Redis is fast, it’s not durable for our use case.
I ate apples because I was hungry.

阅读长难句

如何分析长难句：去修饰找核心
1、把握核心意思，通过逗号和连接词断开句子，通过连接词找到从句，通过谓语动词找到真正主句
2、观察非谓语动词，从句，介词短语等修饰是那个句子成分的
3、一个长句子表达的一个综合的意思，划分一个为意思群

造句技巧

句子格式： When + Who/Which/What(主) + action(谓，动词变化) + Who/Which/What(宾) + How + Where + When

造句训练方法：
1、动词变化：动词的时态/情态
2、四句型转换：肯定句，否定句，一般疑问句，特殊疑问句,There be,系表
3、其他单词+词性的组合排列
